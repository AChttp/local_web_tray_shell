# Switch v1.0.2

Maintenance release focused on embedded web navigation and desktop shell behavior.

## Highlights

- Links that request a new window now open in the current embedded WebView.
- The site section now includes `返回` and `主页` controls for page history and the configured site URL.
- Back navigation uses WebView2 history first and falls back to the app's recorded page history when needed.
- Failed page navigation now reports the actual target URL.
- Borderless window taskbar clicks now support the expected foreground-window minimize behavior.

## Included In This Release

- `Switch.exe`
- `Switch-v1.0.2-win-x64.zip`

## Runtime Requirement

- Microsoft WebView2 Runtime must be installed on the target Windows machine
