# Switch v1.0.3

Maintenance release focused on custom title bar window behavior.

## Highlights

- Double-clicking the title bar now toggles consistently between normal and maximized window states.
- Title bar dragging still behaves normally, but system move now starts only after the pointer crosses the drag threshold.

## Included In This Release

- `Switch.exe`
- `Switch-v1.0.3-win-x64.zip`

## Runtime Requirement

- Microsoft WebView2 Runtime must be installed on the target Windows machine
