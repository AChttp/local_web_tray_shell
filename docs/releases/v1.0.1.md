# Switch v1.0.1

Maintenance release focused on sidebar resizing responsiveness and command log rendering performance.

## Highlights

- The left control panel now uses a single owner-drawn surface instead of the previous mixed control tree.
- Sidebar drag resizing updates the left panel and right workspace together during the drag frame.
- Command log updates are batched and appended incrementally where possible, reducing UI work during noisy command output.
- Obsolete sidebar list/card controls were removed to simplify future maintenance.

## Included In This Release

- `Switch.exe`
- `Switch-v1.0.1-win-x64.zip`

## Runtime Requirement

- Microsoft WebView2 Runtime must be installed on the target Windows machine
