# Switch v1.0.0

First public release of `Switch`.

## Highlights

- Manage multiple local web apps from one Windows desktop shell
- Switch between embedded web pages and command logs in the same workspace
- Add, edit, delete, and launch local commands from the left control panel
- Collapse to the system tray and restore quickly when needed
- Chinese-first UI with a custom Switch icon

## Included In This Release

- `Switch.exe`
- `Switch-v1.0.0-win-x64.zip`

## Runtime Requirement

- Microsoft WebView2 Runtime must be installed on the target Windows machine
